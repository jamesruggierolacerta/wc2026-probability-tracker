# World Cup 2026 Probability Tracker (GitHub Pages)

This version is optimised for iPhone:
- Outputs and Results Entry switch to a **card layout** on small screens (no horizontal scrolling).
- Desktop keeps full tables.

## Deploy
1. Upload the contents of this zip to your GitHub repo root.
2. Settings → Pages → Deploy from branch → main → /(root).

## Notes
- Probabilities are for analysis/tracking.
